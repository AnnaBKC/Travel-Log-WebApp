# Travel-Log-WebApp

## Overview
- Travel Blog web app is a platform in which users are able to share their travel experiences and their personal insights and suggestions. 
- Users are able to register, log in, see their profile information, make posts detailing their experiences, and see other posts made by other users.

## Technology Stack
- MERN stack (MongoDB, Express.js, Node.js, React)
- Javascript, HTML, CSS
- Bootstrap 
- Mongoose library to create schemas and perform back-end queries to the database

## Application

### Home Page
- The user experience starts at the home page, in which a user can visualize all the posts created by users detailing their traveling experiences. 
- These posts relate information about travel destinations, reviews, recommendations, or any other facts and ideas that they want people to know before going on the trip.
- Users are able to search for posts by author, location, title, and date and filter posts by Recent Posts or Oldest Posts.
