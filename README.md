# Travel-Log-WebApp

## Overview
- Travel Blog web app is a platform in which users are able to share their travel experiences and their personal insights and suggestions. 
- Users are able to register, log in, see their profile information, make posts detailing their experiences, and see other posts made by other users.

## Technology Stack
- MERN stack (MongoDB, Express.js, Node.js, React)
- Javascript, HTML, CSS
- Bootstrap 
- Mongoose library to create schemas and perform back-end queries to the database
